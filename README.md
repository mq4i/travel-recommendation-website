## Travelix | Travel Recommendation Website

Description
-I developed a travel recommendation website that helps users explore and discover destinations based on their interests. The website includes a clean navigation bar, search functionality, and sections such as Home, About, and Contact.

How to Run
1. Download the files
2. Open the project folder
3. Run the index.html file in your browser

Features
- HTML Structure & Layout
- CSS Styling & Responsive Design
- Navigation Bar & UI Design
- Search Bar Implementation
- JavaScript Functionality
- DOM Manipulation & Event Handling 

Requirements
- Web Browser (Chrome, Edge, Firefox, etc.)
